After a long while of silence and with loads of new features we are excited to
announce the release of Metals v0.8.0 codename `Cobalt`. This release includes a
huge number of new features and improvements.

Metals is a language server for Scala that works with VS Code, Atom, Vim,
Sublime Text and Emacs. Metals is developed at the
[Scala Center](https://scala.epfl.ch/) and [VirtusLab](https://virtuslab.com)
along with contributors from the community.

In this release we merged 201 PRs and closed 24 issues, full details:
https://github.com/scalameta/metals/milestone/9?closed=1

## TL;DR

Check out the website and give Metals a try: https://scalameta.org/metals/

- goto implementation
- rename symbol
- support for Pants build tool
- run and debug for tests and main methods
- breakpoints
- improvements for Bloop integration
- first code action - import missing symbol
- worksheet support
- Add completion to add all abstract members To mention
